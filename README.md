## Step by Step Using

1. Edit the .env file with the database settings
2. Start the database
3. Install Laravel UI

```bash
    composer -vvv require laravel/ui
```

4. Install the ui for laravel

```bash
    php artisan ui vue --auth
```

5. Start Node
```bash
    npm install && npm run dev
```
6. Start PHP Migrate

```bash
    php artisan migrate
```
